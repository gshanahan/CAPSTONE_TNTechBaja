# CAPSTONE_TNTechBaja
This serves as a repository for the code created for my Capstone Software Engineering project. 

About This Project:
TNTech Baja Truck Sensor Array for Testing and Optimization. 
This project was in collaboration with the TNTech Baja SAE Team.
 - Designing and configuring an on-board sensor array for a Baja Race Buggy, 
    for data acquisition, telemetry for design validation, and potential 
    optimization of suspension design, frame design, and weight reduction.
 - Hardware and embedded system design.
